# OCB MicroSplat Snow Demo Mod - Addon for OCB MicroSplat

Spruce up vanilla snow biome by mixing in another texture.

Requires [OcbMicroSplat][1] (Download from [Nexusmods][3]).

You need to disable EAC to use this mod!
